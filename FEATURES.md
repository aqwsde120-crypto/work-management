# 기능 상세 문서 📖

## 전체 기능 목록

### 1. 📊 대시보드

#### 주요 메트릭 카드
- **전체 프로젝트 수**: 데이터베이스의 모든 프로젝트 개수
- **진행 중**: 상태가 "진행 중"인 프로젝트 수
- **완료**: 상태가 "완료"인 프로젝트 수
- **평균 진척률**: 모든 프로젝트의 진척률 평균값

#### 시각화 차트
**담당자별 워크로드 차트**
- 각 담당자가 맡은 프로젝트 수를 막대 그래프로 표시
- Plotly 인터랙티브 차트 (확대/축소 가능)
- 색상 그라데이션으로 워크로드 시각화

**상태별 분포 파이 차트**
- 진행 중 (파란색)
- 완료 (녹색)
- 일정 지연 (빨간색)
- 검토 중 (주황색)
- 도넛 차트 형태로 표시

#### 경고 및 알림
**지연 프로젝트**
- 상태가 "일정 지연"인 모든 프로젝트 나열
- 프로젝트명, 업무 제목, 마감일 표시
- 빨간색 경고 메시지

**마감 임박 프로젝트 (7일 이내)**
- 현재 날짜 기준 7일 이내 마감 프로젝트
- 완료된 프로젝트는 제외
- 남은 일수 (D-N) 표시
- 노란색 경고 메시지

### 2. 📋 프로젝트 테이블

#### 테이블 컬럼
1. **프로젝트명**: 프로젝트의 큰 범주
2. **업무 제목**: 구체적인 작업 내용
3. **담당자**: 쉼표로 구분된 담당자 목록
4. **분류**: MD #1, MD #2, GTM 등 카테고리
5. **진행 현황**: 색상 코딩된 상태
6. **계획 일정**: 0-100% 진행률 바
7. **실제 진행**: 0-100% 진행률 바
8. **프로젝트 진척률**: 0-100% 진행률 바 + 퍼센트
9. **마감일**: YYYY-MM-DD 형식
10. **승인**: 체크박스 (✅/❌)

#### 필터 기능
**프로젝트명 검색**
- 프로젝트명 또는 업무 제목에서 텍스트 검색
- 대소문자 구분 없음
- 실시간 필터링

**담당자 멀티셀렉트**
- 여러 담당자 동시 선택 가능
- 선택한 담당자 중 하나라도 포함된 프로젝트 표시

**상태 필터**
- 진행 중, 완료, 일정 지연, 검토 중
- 다중 선택 가능

**분류 필터**
- MD #1, MD #2, GTM, 기타
- 다중 선택 가능

#### Excel 내보내기
- 현재 필터링된 데이터를 Excel 파일로 다운로드
- 파일명: `프로젝트_관리_YYYYMMDD.xlsx`
- 모든 컬럼 포함
- 날짜 형식 자동 변환

#### 테이블 스타일
- 진행률 바: 파란색 그라데이션
- 승인 체크박스: 클릭 가능
- 반응형 디자인: 자동 너비 조정
- 스크롤 가능: 600px 고정 높이

### 3. 🗂️ Kanban 보드

#### 4개 컬럼 구조
1. **할 일**: 검토 중인 프로젝트
2. **진행 중**: 진행 중인 프로젝트
3. **일정 지연**: 지연된 프로젝트
4. **완료**: 완료된 프로젝트

#### 카드 정보
각 카드에 표시되는 정보:
- 프로젝트명 (작은 글씨)
- 업무 제목 (큰 글씨)
- 담당자 (👤 아이콘)
- 마감일 (📅 아이콘 + 날짜)
- 마감일 상태 표시:
  - 🔴 지연됨 (마감일 경과)
  - 🟡 임박 (7일 이내)
  - 🟢 여유 (7일 이상)
- 진행률 바 + 퍼센트

#### 인터랙션
- 호버 효과: 마우스 오버 시 카드 확대
- 시각적 피드백: 그림자 효과
- 왼쪽 컬러 바: 프로젝트 카테고리 구분

### 4. ➕ 새 프로젝트/업무 추가

#### 입력 필드
**왼쪽 컬럼:**
- **프로젝트명** (필수): 텍스트 입력
- **업무 제목** (필수): 텍스트 입력
- **업무 설명**: 멀티라인 텍스트 영역
- **담당자** (필수): 멀티셀렉트 (여러 명 선택 가능)

**오른쪽 컬럼:**
- **분류** (필수): 드롭다운 (MD #1, MD #2, GTM, 기타)
- **진행 현황** (필수): 드롭다운 (진행 중, 완료, 일정 지연, 검토 중)
- **계획 일정**: 슬라이더 (0-100%)
- **실제 진행**: 슬라이더 (0-100%)
- **진척률**: 슬라이더 (0-100%)
- **마감일** (필수): 날짜 선택기 (기본값: 30일 후)
- **승인됨**: 체크박스

#### 유효성 검사
- 필수 항목(*) 미입력 시 에러 메시지
- 성공 시 축하 풍선 애니메이션
- 자동으로 프로젝트 테이블로 이동

#### 데이터베이스 저장
- SQLite INSERT 쿼리 실행
- 생성 시간 자동 기록 (created_at)
- 고유 ID 자동 생성 (AUTO_INCREMENT)

---

### 5. 👥 팀원 관리

#### 현재 팀원 목록
- 이모지 + 이름 표시
- 각 팀원 옆에 삭제 버튼
- 알파벳/한글 순 정렬

#### 팀원 추가
**입력 필드:**
- **이름**: 텍스트 입력 (고유값, 중복 불가)
- **이모지**: 텍스트 입력 (기본값: 👤)

**유효성 검사:**
- 이름 중복 확인
- 빈 이름 입력 방지

#### 팀원 삭제
- 즉시 삭제 (확인 프롬프트 없음)
- 삭제 후 자동 새로고침
- 데이터베이스에서 영구 삭제

**주의사항:**
- 팀원 삭제 시 해당 담당자가 할당된 프로젝트는 그대로 유지됨
- 담당자 필드는 텍스트로 저장되어 있음

## 기술적 특징

### 데이터베이스 구조

**tasks 테이블**
```sql
- id: INTEGER PRIMARY KEY AUTOINCREMENT
- project_name: TEXT NOT NULL
- title: TEXT NOT NULL
- description: TEXT
- assignee: TEXT (쉼표로 구분)
- category: TEXT
- status: TEXT
- planned_progress: INTEGER (0-100)
- actual_progress: INTEGER (0-100)
- completion_rate: INTEGER (0-100)
- deadline: DATE
- approved: INTEGER (0 또는 1)
- created_at: TIMESTAMP (자동 생성)
```

**team_members 테이블**
```sql
- id: INTEGER PRIMARY KEY AUTOINCREMENT
- name: TEXT UNIQUE NOT NULL
- emoji: TEXT (기본값: '👤')
```

### 성능 최적화
- SQLite 로컬 데이터베이스 사용
- 쿼리 결과 Pandas DataFrame으로 캐싱
- 필터링은 Pandas로 인메모리 처리
- 페이지 간 데이터베이스 연결 재사용

### 보안 기능
- 세션 기반 비밀번호 인증
- Secrets 파일로 민감 정보 분리
- SQL Injection 방지 (parameterized queries)
- XSS 방지 (Streamlit 자동 이스케이핑)

### UI/UX 특징
- 반응형 디자인 (모바일 최적화)
- 커스텀 CSS로 프로페셔널한 스타일
- 다크모드 지원 (Streamlit 네이티브)
- 색상 코딩으로 정보 전달
- 인터랙티브 차트 (Plotly)
- 실시간 필터링 및 검색

### 확장성
- 모듈화된 함수 구조
- 새 기능 추가 용이
- 데이터베이스 스키마 확장 가능
- 커스텀 컬럼 추가 가능

## 사용 시나리오

### 시나리오 1: 신규 프로젝트 등록
1. "➕ 새 프로젝트/업무 추가" 메뉴 선택
2. 필수 정보 입력
3. 담당자 2명 지정
4. 마감일 설정
5. "프로젝트 추가" 버튼 클릭
6. 대시보드에서 즉시 확인

### 시나리오 2: 진행 상황 모니터링
1. "📊 대시보드" 메뉴에서 전체 현황 확인
2. "지연 프로젝트" 섹션에서 문제 파악
3. "📋 프로젝트 테이블"에서 상세 데이터 분석
4. 담당자별 필터링으로 개인 워크로드 확인

### 시나리오 3: 주간 보고서 작성
1. "📋 프로젝트 테이블"에서 상태 필터 적용
2. 완료된 프로젝트만 선택
3. "Excel 다운로드" 버튼으로 데이터 추출
4. Excel 파일로 보고서 작성

### 시나리오 4: 칸반 보드로 스프린트 관리
1. "🗂️ Kanban 보드" 메뉴 선택
2. "할 일" 컬럼에서 다음 작업 확인
3. 각 카드의 진행률 점검
4. 마감 임박 프로젝트 우선순위 설정
