## 🚀 즉시 시작하기 (3단계)

### Step 1: 파일 준비 ✅
모든 파일을 하나의 폴더에 다운로드했습니다.

### Step 2: 의존성 설치 📦
터미널/명령 프롬프트에서:
```bash
cd [다운로드한 폴더]
pip install -r requirements.txt
```

### Step 3: 앱 실행 🎯
```bash
streamlit run app.py
```

브라우저가 자동으로 열리면 성공!
- **로그인 비밀번호**: `team123`

### 프론트엔드

- **커스텀 CSS**: 프로페셔널 스타일

- SQL Injection 방지

## 🆘 문제 해결 빠른 참조

| 문제 | 해결 방법 |
|------|----------|
| 앱이 실행되지 않음 | `pip install --upgrade -r requirements.txt` |
| 비밀번호를 잊음 | `.streamlit/secrets.toml` 파일 확인 |
| 데이터가 이상함 | `rm projects.db` 후 재시작 |
| 포트 충돌 | `streamlit run app.py --server.port 8502` |
| 배포 실패 | Secrets 설정 확인 |
